HTTP Content Negotiation

  • Content Negotiation (Inhaltsvereinbarung)

    • - Technik zur Abstimmung der Inhalte der angefragten Ressource auf die Möglichkeiten des Clients

  • Anfrage-Header-Felder

    • - Accept (Liste akzeptierter Inhaltstypen)

    - Gegenstück: Content-Type-Feld

  • Qualitätsparameter "q"
Unter Content Negotiation versteht man im HTTP eine Technik, die eine Abstimmung der Inhalte der angefragten Ressource aufgrund der Möglichkeiten und Vorzüge des Clients ermöglicht - zur Inhaltsvereinbarung stehen im HTTP die Anfrage-Header-Felder : Accept, Accept-Charset, Accept-Encoding, Accept-Language zur Verfügung Accept ist z.B. Liste aller akzeptieren Inhalte, Content-Type-Feld ist Gegenstück zu dieser Anfrage Bewertung durch Qualitätsparameter "q" von 0-1 (optional)
