MIME-Type

  • MIME steht für "Multipurpose Internet Mail Extensions"

    • - Internet Media Type

    - Content Type (Zeichenzusatz und Typ des Inhalts)

  • Standardisierte Angabe zum Format einer Datei

    • - Grundform: "Kategorie/Unterkategorie"

Geschichte

  • Ursprünglich für Multipart-Mails gedacht
  • Konvention zur Trennung der einzelnen Teile
  • MIME-Typen Schema

Zweck und Verwendung

  • Kommunikation und Datenübertragung (Art der Daten)
  • Klassifiziert Daten im Rumpf einer Nachricht
  • Sender und Empfänger müssen sich einigen, ob Empfänger MIME-Typ akzeptiert
  • IANA

    • - (x-"subtype")

  • MIME-Sniffing

HTTP Content Negotiation

  • Content Negotiation (Inhaltsvereinbarung)

    • - Technik zur Abstimmung der Inhalte der angefragten Ressource auf die Möglichkeiten des Clients

  • Anfrage-Header-Felder

    • - Accept-Charset (Liste akzeptierter Zeichensätze)

    - Gegenstück: Content-Type-Feld
